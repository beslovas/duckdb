DuckDB Helm Charts

Welcome to the DuckDB Helm Charts repository. This repository contains Helm charts for deploying DuckDB with UI support, configurable extensions, and persistent storage.

Usage

Helm is required to use and operate these charts. You can add the DuckDB Helm Repository like below:

helm repo add beslovas https://beslovas.github.io/duckdb

Available Charts

DuckDB Chart

A Helm chart for DuckDB with UI support, configurable extensions, and persistent storage.

Install:

helm install my-duckdb beslovas/duckdb

Upgrade:

helm upgrade my-duckdb beslovas/duckdb

Documentation

Documentation and the default values.yaml can be found in the relevant chart repository:

  • DuckDB Helm Chart Repository
  • DuckDB Official Documentation

Features

  • DuckDB UI support
  • Configurable extensions (httpfs, parquet, json, sqlite)
  • Persistent storage support
  • Configurable duckdbrc settings
  • Multi-architecture Docker images
  • Kubernetes-ready deployment
